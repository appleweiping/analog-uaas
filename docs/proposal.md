# Proposal

## Title

Reliable Surrogate-Based Analog Circuit Optimization via Failure-Aware Uncertainty Calibration and Utility-Driven Simulation Scheduling

## Background and Motivation

模拟电路优化本质上是一个高维、黑盒且计算成本昂贵的约束优化问题。给定一组设计变量，例如晶体管尺寸、偏置电流和补偿电容，设计者通常需要通过反复调用 SPICE 级仿真来评估增益、带宽、相位裕度和功耗等关键性能指标，并在满足约束条件的前提下逼近最优设计。传统流程高度依赖工程师经验以及反复试错，其主要瓶颈并不在于缺少优化目标，而在于每一次高保真评估都伴随着显著的时间成本。因此，如何在保证设计质量与可靠性的同时减少仿真调用，始终是模拟电路设计自动化中的核心问题。

近年来，代理模型驱动的优化方法为这一问题提供了重要方向。这类方法通过学习设计参数到性能指标之间的映射关系，用可微、快速的学习模型近似高成本仿真器，并结合采样或优化策略减少真实仿真次数。然而，现有方法普遍将重点放在平均预测精度上，而忽略了一个更关键的问题：在真实优化过程中，决定方法是否可靠的并不是全局平均误差，而是模型在高风险区域，尤其是设计约束边界附近，是否会产生系统性误判。一旦代理模型在靠近边界的候选点上错误地将不可行设计判断为可行，优化过程就会被错误引导，表现为后期震荡、反复调用仿真、甚至收敛失败。

因此，本项目的核心目标不是单纯构建一个更高精度的回归器，而是围绕“代理模型何时值得信任”这一问题，构建一个面向可靠优化的模型—仿真协同框架。该框架将不确定性估计、风险建模、主动采样与仿真调度整合为统一闭环，使系统能够根据当前候选点的预测值、置信度与边界风险动态决定何时依赖代理模型、何时调用真实仿真，从而在控制误判风险的同时尽可能减少仿真预算。

## Research Question

本项目围绕以下三个问题展开。第一，现有代理模型方法在模拟电路优化后期为何会失效，尤其是在接近约束边界时，这种失效是否可以被形式化定义并通过数据加以验证。第二，如何将代理模型输出的不确定性从一个附加诊断信号，升级为优化过程中的核心决策依据，使其能够服务于仿真调度而不仅仅是误差描述。第三，如何设计一个任务导向而非纯不确定性导向的主动采样与闭环更新策略，使模型采集的是“对优化最有价值”的样本，而不是简单地在全空间中追逐高方差区域。

## Problem Definition

设设计变量为 x ∈ X，真实电路性能由高保真仿真器 f(x) 给出，其中输出包括 gain、GBW、phase margin、power 等多个指标。我们希望在给定仿真预算 B 的条件下，求解满足约束集合 C 的可行设计，并在可行域内优化目标函数 g。与传统黑盒优化不同，本项目引入代理模型 \hat{f}(x) 以近似 f(x)，但不假设代理模型在整个设计空间中始终可靠。相反，我们显式考虑代理模型在边界区域的误差放大及可行性误判问题，并将其视为影响优化质量与效率的主要风险源。

## Core Hypothesis

本文假设：代理模型在接近设计约束边界时会表现出显著更高的误差和可行性误判率，而这种误判会在优化后期累积并系统性改变搜索轨迹。如果能够通过经校准的不确定性建模与误判风险建模识别此类高风险候选点，并在这些区域自适应增加真实仿真调用，则可以在不显著增加总预算的前提下，提升优化过程的稳定性、可行解命中率和整体效率。

## Proposed Method

本项目提出一个 failure-aware 的 surrogate-assisted optimization 框架。系统首先在固定拓扑与固定工艺条件下，对两级运放设计空间进行初始采样并生成结构化仿真数据集。随后训练多输出代理模型作为性能预测器，初始实现采用 MLP ensemble，并以 ensemble variance 近似 epistemic uncertainty。与传统方法不同，本项目不将不确定性直接作为采样分数，而是进一步引入 uncertainty calibration，使预测方差与真实误差统计更一致，再结合约束边界距离构造可行性误判风险。基于这一风险量，系统对候选点执行自适应决策：在低风险区域优先使用代理模型进行快速搜索，在高风险区域强制调用真实仿真，并将新样本反馈到数据池中进行在线更新。

在此基础上，主动学习策略也从单纯的 uncertainty-driven sampling 升级为 utility-driven sampling。也就是说，系统不仅考虑候选点是否不确定，还考虑其对当前优化目标是否具有实际价值。一个远离可行域但方差很大的点，对优化可能是无意义的；一个逼近最优解、且处于边界敏感区域的中等不确定性点，反而更值得优先仿真。最终，整个方法形成“候选生成—代理预测—不确定性校准—误判风险评估—仿真调度—数据回流—模型更新”的闭环。

## Expected Contributions

本项目的第一项预期贡献，是系统性地分析并量化代理模型在模拟电路优化后期的失效模式，特别是其在接近约束边界时的误差分布偏移与可行性误判现象。第二项贡献，是提出一个经校准的不确定性感知、面向边界风险的仿真调度框架，使真实仿真调用从固定预算或静态规则转变为由风险与效用驱动的动态行为。第三项贡献，是构建一个完整且可复现的模型—仿真协同优化系统，将数据生成、代理建模、风险分析、主动学习和评测统一到同一实验平台中。第四项贡献，是在典型模拟电路设计任务以及困难约束场景下，证明该方法能够在保持或提高设计成功率的同时显著减少仿真次数，并提升后期优化阶段的稳定性与可靠性。

## Scope and Target Setting

为确保问题定义尖锐且实验闭环完整，本项目在第一篇工作中固定单一拓扑，优先选择两级运放作为研究对象，而不追求多拓扑泛化。这一选择的原因在于，两级运放具备较为标准的 gain、GBW、phase margin 与 power 之间的 trade-off 关系，特别适合构造边界区 failure case，并验证可靠优化框架的有效性。跨拓扑泛化、更加复杂的结构决策与 agent-level 设计策略，将作为后续工作的自然扩展，而非当前工作的主目标。

## Deliverables

本项目最终将交付以下成果：一个结构化、可复现的模拟电路代理优化代码仓库；一个包含仿真数据、边界标签和评测协议的数据与实验系统；一组围绕 failure mode、不确定性质量、主结果、困难场景和 wall-clock 成本的核心实验图表；以及一篇围绕可靠 surrogate-based analog optimization 的顶会风格论文初稿。
